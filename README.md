# dotfiles

Collection of scripts and dotfiles to setup my env

## Install Instructions

Clone the repository and install the plugins:

```sh
mkdir ~/workspace && cd ~/workspace
git clone git@github.com:vmule/dotfiles
```

Run setup.sh:

```sh
bash ./setup.sh
```
